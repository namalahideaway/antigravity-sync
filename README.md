# Antigravity Sync

Export, import, and sync all your Antigravity IDE data between computers using GitHub.

## Features

- **Export** all AG data to a portable ZIP file (chats, brain, knowledge, projects, config)
- **Import** from ZIP with conflict resolution (overwrite, skip, newer-wins)
- **GitHub Sync** — push/pull your data to a private GitHub repo
- **Dashboard** — premium webview UI with status, category toggles, and sync history
- **Selective Sync** — include/exclude any category (12 categories)
- **Auto-Sync** — automatic periodic sync to GitHub
- **Status Bar** — real-time sync status indicator

## Categories

| Category | What's Included | Default |
|----------|----------------|---------|
| Chats | Conversation history | ✅ |
| Brain | Plans, walkthroughs, artifacts | ✅ |
| Knowledge | Distilled KI topics | ✅ |
| Projects | Scratch project files | ✅ |
| Config | Settings, MCP, GEMINI.md | ✅ |
| Recordings | Browser recordings | ❌ (large) |
| + 6 more | Annotations, prompting, etc. | ✅ |

## Commands

- `AG Sync: Export Data to ZIP` (Ctrl+Alt+Shift+E)
- `AG Sync: Import Data from ZIP`
- `AG Sync: Open Dashboard` (Ctrl+Alt+Shift+S)
- `AG Sync: Push to GitHub` (Ctrl+Alt+Shift+P)
- `AG Sync: Pull from GitHub`
- `AG Sync: Full Sync`
- `AG Sync: Connect GitHub Account`
- `AG Sync: Select/Create Sync Repo`
- `AG Sync: Select Categories`
- `AG Sync: View Changes`
- `AG Sync: Toggle Auto-Sync`
- `AG Sync: Scan Data`
- `AG Sync: Show Status`
- `AG Sync: Reset Sync State`

## Quick Start

1. Install the extension
2. Run `AG Sync: Open Dashboard` (Ctrl+Alt+Shift+S)
3. Connect your GitHub account
4. Select or create a sync repo
5. Push your data!
